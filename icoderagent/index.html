What it does

A genuine agent loop — not just a chat box.

🤖

Autonomous agent

Give it a task and it plans, writes files, runs code, reads the results and iterates — right on your device.

🐍

Real Python, embedded

A full CPython runtime is built in, so the agent can actually execute the code it writes — no server round-trip.

🧩

Your choice of model

Works with Anthropic Claude, OpenAI GPT, xAI Grok, Apple MLX and other local models. Switch providers anytime.

💻

Mac companion

Pair with the desktop companion to drive a real project on your Mac from your phone, over a secure relay.

Design & build

Generate HTML/Markdown, preview your work, and sketch flows — a lightweight design surface built in.

📉

Token saver

On-device output compaction trims tool noise before it reaches the model, so you spend fewer tokens per task.

Private by design

Your code and keys never touch our servers.

🔐

Keys in the Keychain

Your API keys are stored in the iOS Keychain and sent only to the provider you chose — never to us.

📦

Local files & history

Projects, chat history and usage logs stay on your device. Delete the app and they're gone.

🚫

No trackers

No advertising SDKs, no third-party analytics, no cross-app tracking. Read the privacy policy.
